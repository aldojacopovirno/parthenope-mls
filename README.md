Official website repository for the Parthenope Machine Learning Society (PMLS). A student organization dedicated to promoting education, research, and community engagement in machine learning and artificial intelligence.

**Live Site:** [https://aldojacopovirno.github.io/parthenope-mls/](https://aldojacopovirno.github.io/parthenope-mls/)

## Structure

```
parthenope-mls/
├── components/
│   ├── About.tsx              # About section component
│   ├── BlogSection.tsx        # Blog posts grid
│   ├── BlogPostDetail.tsx     # Individual blog post view
│   ├── ContactSection.tsx     # Contact form and info
│   ├── Footer.tsx             # Site footer with social links
│   ├── Header.tsx             # Navigation header
│   ├── Hero.tsx               # Landing hero section
│   ├── PrivacyPolicy.tsx      # Privacy policy page
│   ├── TeamSection.tsx        # Team members display
│   └── TermsOfService.tsx     # Terms of service page
├── blog-posts/
│   └── [Blog post content files]
├── public/
│   ├── images/                # Static image assets
│   └── .nojekyll              # Disables Jekyll processing on GitHub Pages
├── styles/
│   └── blog-post.css          # Blog post styling
├── App.tsx                    # Main application component
├── index.html                 # HTML entry point
├── index.tsx                  # React entry point
├── types.ts                   # TypeScript type definitions
├── metadata.json              # Site metadata
├── vite.config.ts             # Vite configuration
└── tsconfig.json              # TypeScript configuration
```
